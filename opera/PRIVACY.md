# Privacy Policy - View Page Source Opera Extension

**Last Updated: December 29, 2025**

## Overview

View Page Source ("the Extension") is committed to protecting your privacy. This privacy policy explains what data is collected, how it's used, and your rights regarding your information.

## Developer Information

- **Extension Name**: View Page Source
- **Website**: https://www.view-page-source.com
- **Contact**: Available through the website

## Data Collection and Usage

### What Data the Extension Collects

The Extension collects and processes **minimal data** required for functionality:

1. **Current Tab URL** (only when you actively use the extension)
   - **When**: Only when you click the extension icon, use the context menu, or use the keyboard shortcut
   - **Why**: To send the URL to the View Page Source website for analysis
   - **How Long**: Not stored by the extension; immediately sent to the website
   - **Permission**: `activeTab` (Opera permission)

2. **User Preferences** (optional settings)
   - **What**: Your syntax highlighting and word wrap preferences
   - **Where**: Stored locally in Opera's sync storage
   - **Why**: To remember your settings across devices
   - **How Long**: Until you uninstall the extension or clear browser data
   - **Permission**: `storage` (Opera permission)

### What Data is NOT Collected

The Extension does **NOT** collect or transmit:
- Browsing history
- Personal information (name, email, etc.)
- Cookies or authentication tokens
- Form data or passwords
- Contents of web pages you visit (unless you explicitly request source code viewing)
- Usage analytics or telemetry
- IP addresses (collected by the website, not the extension)

## Data Sharing and Third Parties

### View Page Source Website

When you use the extension, the URL you choose to analyze is sent to:
- **URL**: https://www.view-page-source.com
- **Purpose**: To fetch and analyze the webpage's source code
- **What's Sent**: Only the URL you explicitly choose to analyze
- **What's Received**:
  - HTML source code with syntax highlighting
  - Performance metrics (DNS, TCP, TLS, server response, download times)
  - Server information (HTTP version, status code, headers)
  - Smart Analysis data (SEO meta tags, Open Graph, structured data, links, resources, security indicators)
  - Technology Detection results (CMS, frameworks, analytics, CDNs, etc.)

The View Page Source website has its own privacy policy available at: https://www.view-page-source.com/privacy

### No Third-Party Analytics

The Extension itself does **NOT**:
- Use Google Analytics
- Use any third-party tracking services
- Send data to any third parties
- Display advertisements

## Permissions Explanation

The Extension requests the following Opera permissions:

### activeTab
- **Purpose**: Access the URL of the current tab only when you interact with the extension
- **Scope**: Limited to the active tab, only when you click/invoke the extension
- **Not Used For**: Browsing history, tracking, or accessing tab content

### contextMenus
- **Purpose**: Add "View Page Source" option to the right-click menu
- **Scope**: Creates a menu item in the browser's context menu
- **Not Used For**: Any data collection or tracking

### storage
- **Purpose**: Save your preferences (syntax highlighting, word wrap settings)
- **Scope**: Opera's sync storage for preference persistence
- **Not Used For**: Storing browsing data or personal information

## User Rights and Control

### Your Choices

You have complete control over the Extension:

1. **Choose What to Analyze**: The extension only sends URLs you explicitly choose to view
2. **Disable at Any Time**: Disable or uninstall the extension from `opera://extensions`
3. **Clear Settings**: Uninstalling the extension removes all stored preferences
4. **No Account Required**: No registration, login, or account creation needed

### Opt-Out

To stop all data processing by the Extension:
1. Go to `opera://extensions`
2. Disable or remove "View Page Source"
3. Your preferences will be deleted upon uninstall

## Data Security

### Extension Security
- Code is open source and auditable
- No external scripts loaded by the extension
- Uses secure HTTPS for all website communication
- Follows Opera's security best practices
- Regular security updates

### Website Security
The View Page Source website (where URLs are sent):
- Uses HTTPS encryption (TLS 1.3)
- Implements rate limiting to prevent abuse
- Validates and sanitizes all inputs
- Does not store source code permanently (cached for 2 hours)

## Children's Privacy

The Extension does not knowingly collect information from children under 13. The Extension is intended for developers, designers, and technical users. No age verification is required as no personal information is collected.
